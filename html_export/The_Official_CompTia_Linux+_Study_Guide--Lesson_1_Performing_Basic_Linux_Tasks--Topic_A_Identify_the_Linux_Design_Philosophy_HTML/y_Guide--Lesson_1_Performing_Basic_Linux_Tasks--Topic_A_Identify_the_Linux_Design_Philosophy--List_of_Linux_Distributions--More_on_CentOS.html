More on CentOS


The CentOS Linux distribution is a stable, predictable, manageable, and reproducible platform derived from the sources of RHEL. CentOS is maintained by the CentOS Project, a community-driven free software effort that has its own governing board. The members of the CentOS Project work independently of the RHEL team. However, CentOS benefits from Red Hat's ongoing contributions and investment, and the CentOS trademark is owned by Red Hat.

This course uses CentOS because it provides a free enterprise-class computing platform that aims to be functionally compatible with the upstream product (RHEL) that it derives from. CentOS does not contain Red Hat's product or certifications, although it is build from the same sources as the upstream enterprise products. More details about this are available in the CentOS FAQ here:

https://wiki.centos.org/FAQ/General

For production environments, the licensed and fully supported RHEL produt is recommended.
